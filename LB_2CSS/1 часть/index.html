Свойство CSS overflow определяет, что будет с элементом, если его содержимое не помещается в видимые пределы элемента.

overflow:scroll

Вы можете использовать свойство overflow для контроля над разметкой. На некоторых сайтах используются окошки с меню и полосами прокрутки. Значение по умолчанию visible.
